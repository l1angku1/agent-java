## 🚀 3. 快速上手指南
1. **首次清洗**：用中性洗涤剂与温水清洗杯体及杯盖，擦干备用。
2. **充电激活**：插入 Type-C 线，指示灯红→蓝闪烁表示充电中，常亮表示满电（约2.5小时）。
3. **下载APP**：扫描说明书背面二维码，或应用商店搜索 `智享生活` 安装。
4. **蓝牙配对**：长按杯底电源键3秒开机 → 打开APP点击 `添加设备` → 靠近杯体自动发现 → 按提示完成绑定。
5. **设置温度**：APP内选择饮品类型（咖啡/茶/牛奶/自定义），或直接在杯体触控环滑动调节。

## 🛠 4. 功能详解
### 4.1 触控操作
| 操作方式 | 功能响应 |
|:---|:---|
| 短按 1 次 | 开关机 / 查看当前温度（LED屏显3秒） |
| 长按 2 秒 | 进入/退出 温度锁定模式 |
| 顺时针滑动 | 升高温度（每格+2℃） |
| 逆时针滑动 | 降低温度 |
| 快速双击 | 切换 ℃/℉ 显示 |

### 4.2 APP 智能功能
- 📈 **温度曲线记录**：自动保存近30天饮用数据，生成健康报告
- ⏰ **智能提醒**：久坐喝水提醒 / 饮品最佳饮用期提示
- 🌐 **多设备管理**：一台手机可绑定最多5个杯体，适合家庭/办公共享
- 🔒 **儿童锁**：APP内开启后，触控环将失效，防误触烫伤

### 4.3 安全保护机制
- 🔋 过充/过放/短路三重保护
- 🌡️ 干烧检测：液位低于安全线自动断电
- 📦 倾倒断电：倾斜＞45°持续3秒自动休眠
- 🔄 固件OTA升级：APP推送，免拆机优化算法

## ⚠️ 5. 安全注意事项
- ❌ 严禁将杯体放入微波炉、洗碗机、消毒柜或明火加热
- ❌ 请勿装入碳酸饮料、酒精类或强酸强碱液体（防气压/腐蚀）
- 🔌 充电时请保持充电口干燥，严禁湿手插拔充电线
- 👶 产品含高温部件，儿童需在成人监护下使用
- 📉 电池衰减属正常现象，连续使用1年以上续航可能下降15%~20%
- 🌍 请勿在海拔＞3000m 或温度＜-10℃/＞45℃ 环境下长期使用

## 🧼 6. 清洁与保养
1. **日常清洁**：仅支持手洗。用软布+中性清洁剂擦拭内胆与外壳，禁用钢丝球/强效去污粉。
2. **除垢保养**：每月1次，加入200ml温水+5g食品级柠檬酸，静置30分钟后冲洗。
3. **充电口维护**：使用前确认Type-C口无水分/异物，可用干燥棉签轻拭。
4. **长期存放**：请保持电量在40%~60%，每3个月充电一次以防电池休眠。

## ❓ 7. 常见问题排查 (FAQ)
| 问题现象 | 可能原因 | 解决建议 |
|:---|:---|:---|
| 无法开机/指示灯不亮 | 电池耗尽 / 长期未用进入保护 | 连接充电器30分钟后再尝试长按开机 |
| APP搜索不到设备 | 蓝牙未开启 / 距离过远 / 设备已绑定其他账号 | 关闭手机蓝牙重开；距离杯体＜2米；在APP解绑旧账号 |
| 温度显示偏差大 | 首次使用未校准 / 传感器沾附水渍 | APP内 `设置→传感器校准` 执行3步校准流程 |
| 杯盖漏水 | 硅胶圈老化 / 未旋紧 | 检查密封圈是否错位，顺时针拧紧至“咔哒”声 |
| 充电频繁中断 | 充电头功率不足 / 线缆破损 | 更换5V/2A及以上标准充电头；检查线缆触点 |
