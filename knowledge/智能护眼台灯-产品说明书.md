## 🛠 4. 功能深度解析
### 4.1 智能调光系统
| 模式 | 触发条件 | 效果说明 |
|:---|:---|:---|
| 🌞 自动感光 | 开启"环境光自适应" | 内置高精度光传感器，实时补偿环境光变化，维持桌面照度恒定 |
| 📚 专注模式 | APP定时/手动开启 | 45分钟柔和光+5分钟暖光休息提醒，符合番茄工作法 |
| 🌙 夜间模式 | 22:00-6:00自动启用 | 亮度自动降至≤200lux，色温≤3000K，减少褪黑素抑制 |
| 👶 儿童模式 | APP绑定儿童账号 | 锁定最高亮度/色温上限，防止强光刺激，支持用眼时长统计 |

### 4.2 多端控制方式
📍 本地控制
├─ 触控面板：滑动/点击/长按 多手势识别
├─ 旋钮扩展：可选配磁吸物理旋钮（单独购买）
📍 远程控制
├─ 光语APP：远程开关/定时任务/场景联动/固件升级
├─ 智能音箱："打开台灯""调亮一点""切换到阅读模式"
├─ 智能面板：支持接入米家/华为鸿蒙/苹果HomeKit场景自动化
📍 无网使用
└─ 断电记忆：恢复供电后自动回归上次使用状态

### 4.3 健康护眼技术
- 🔬 **全光谱仿真**：补全480nm~500nm蓝光缺口，光谱连续度接近上午10点自然光
- 💡 **侧发光+导光板**：光线经11层光学膜均匀扩散，无重影、无直射眩光
- 🔄 **高频调光**：2500Hz PWM调光频率，远超人眼感知阈值，彻底消除频闪
- 📐 **非对称光路**：光线主要投向桌面工作区，减少屏幕反光干扰

## ⚠️ 5. 安全使用规范
- 🔌 请使用原装电源适配器，严禁使用非标充电器或改装电路
- 💧 灯体仅支持干布擦拭，禁止液体直接接触电路部分（底座接口处）
- 🔥 远离热源/火源，灯头工作温度约45℃，请勿覆盖散热孔
- 👀 避免长时间直视发光面，建议保持30cm以上观看距离
- 🧒 3岁以下儿童使用时需成人陪同，防止拉扯灯臂造成倾倒
- 🌐 如长时间外出，建议通过APP远程关闭电源或启用"离家模式"

## ❓ 7. 常见问题自助排查
| 问题 | 可能原因 | 解决方案 |
|:---|:---|:---|
| 台灯不亮/无反应 | 电源未接通 / 适配器故障 | 检查插座→重插电源线→尝试更换同规格适配器测试 |
| 触控不灵敏 | 表面有水渍/油污 / 静电干扰 | 用干布清洁触控区→断电30秒重启→避免在干燥环境快速摩擦 |
| APP连接失败 | Wi-Fi密码错误 / 仅支持2.4G网络 / 路由器隔离设置 | 确认连接2.4GHz Wi-Fi→关闭"双频合一"→检查路由器AP隔离是否开启 |
| 语音控制无响应 | 音箱未登录同一账号 / 设备未授权 | 在音箱APP中检查设备共享状态→重新授权"光语智能"技能 |
| 自动调光异常 | 光传感器被遮挡 / 环境光剧烈波动 | 检查灯头侧面传感器是否清洁→避免台灯正对窗户/强光源 |
| 色温/亮度调节卡顿 | 固件版本过旧 / 网络延迟 | APP内检查固件更新→尝试本地触控操作对比判断 |
